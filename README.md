## 📝 Project Abstract
The **Savora Restaurant Management Website** is a comprehensive full-stack web application designed to bridge the gap between traditional restaurant operations and modern digital efficiency. Developed as a BCA Capstone project, the system provides an integrated platform for customers to make table reservations and for administrators to monitor performance via an analytics dashboard. By leveraging **ADO.NET** for direct database communication, the application ensures high performance and strict data integrity.

## 📦 System Modules

### 👤 Customer Module
* **Dynamic Landing Page:** Showcases restaurant heritage and menu highlights.
* **Reservation System:** User-friendly interface for booking tables with real-time data persistence.
* **Gallery & About:** Visual storytelling and brand information to engage users.

### 🔐 Admin Module
* **Authentication:** Secure login for restaurant staff and owners.
* **Analytics Dashboard:** Data-driven views showing total reservations and customer trends.
* **Content Management:** Tools to update menu information and manage customer inquiries.

---

## 📂 Database Design (Schema)
The backend is supported by a relational database in SQL Server. Key tables include:
* **Users:** Stores credentials and contact info for administrators.
* **Reservations:** Tracks `BookingID`, `CustomerName`, `PartySize`, `DateTime`, and `SpecialRequests`.
* **Menu:** Contains item names, descriptions, pricing, and categories.

---

## ⚙️ Methodology
1.  **Requirement Gathering:** Identifying pain points of manual restaurant booking.
2.  **UI/UX Design:** Creating high-fidelity logos and layouts using **Inkscape**.
3.  **Database Implementation:** Writing stored procedures and SQL scripts.
4.  **Backend Development:** Implementing logic in C# and connecting to SQL Server via **ADO.NET**.
5.  **Testing & Deployment:** Debugging via Visual Studio 2022 and managing versions through GitHub.

---

## ✨ Key Advantages
* **Direct Data Control:** ADO.NET provides granular control over database queries compared to automated ORMs.
* **Scalability:** The modular structure of Razor Pages allows for easy future updates.
* **Professional Branding:** Integrated custom graphics ensure the project meets industry standards.
